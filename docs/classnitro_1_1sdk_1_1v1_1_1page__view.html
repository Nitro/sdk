◆ set_clip_box()

rect nitro::sdk::v1::page_view::set_clip_box ( rect  box)

Sets the clipping box for the page.

The clip box is further clipped to the page dimensions. The margins are returned.

A clip box can be used to limit rendering to parts of the page. For example one might only want to render the lower half of the page.

If not set, the view's clip box covers the entire page.

◆ set_dpi()

page_view& nitro::sdk::v1::page_view::set_dpi ( double  dpi)

Change the DPI and increase or decrease the size of the crop box to match.

The default DPI for a new page_view is 72, which maps each standard PDF unit (1/72nd of an inch) to a single pixel.

For example, a page with 300x500 PDF units size is rendered as 300x500 pixels with dpi=72. If dpi=144 then the page is rendered at 600x1000 pixels.

Adjusts the view so that the page fits in a window of the given size.

This will modify the DPI and crop box of the view according to the fit:

  • For fit_type::page, the entire page will fit in the window
  • For fit_type::width and fit_type::height, the page will fill the corresponding dimension
